# Sistema de Cadastro de Usuários em Java — Versão 1

> Primeira versão de um sistema de gerenciamento de usuários desenvolvido em Java com foco em Programação Orientada a Objetos, padrão DAO e validação de regras de negócio.

## Sobre o projeto

Este projeto implementa as operações fundamentais de um sistema CRUD de usuários simples, com separação de responsabilidades entre classes e validação de regras de negócio. É a versão inicial de uma série de melhorias progressivas — cada versão evolui a arquitetura e as funcionalidades.

**Objetivo técnico:** praticar POO na prática, não apenas na teoria — modelando entidades reais (como `UserModel`) com comportamentos bem definidos, e aplicando o padrão de projeto **DAO** para separar a lógica de acesso a dados da lógica de negócio.

---

## Funcionalidades

- Cadastro de usuários com nome, e-mail e data de nascimento
- Atualização de dados de um usuário existente por ID
- Exclusão de usuário por ID
- Busca de usuário por ID com tratamento de exceção customizada
- Listagem de todos os usuários cadastrados
- Menu interativo via terminal com validação de opção
